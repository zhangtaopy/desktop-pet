# Desktop Pet - 功能扩展方向

## 一、实用功能

### 1. 番茄钟 / 专注计时器
- 工作时宠物进入"陪伴工作"状态
- 休息时宠物欢呼庆祝
- 自定义工作时长和休息时长

### 2. 久坐 / 休息提醒
- 监测活跃状态，长时间工作后气泡提醒活动
- 生气表情催促

### 3. 天气联动
- 外观和行为随天气变化（下雨打伞、晴天墨镜、下雪围巾）
- 对接天气 API，自动获取位置天气

### 4. 简易待办 / 便签
- 点击宠物弹出迷你便签，快速记录待办
- 宠物根据完成情况表现开心或沮丧

### 5. 系统状态感知
- CPU/内存感知：过热流汗、资源紧张焦虑、正常悠闲

### 6. 剪贴板管理
- 快捷入口展示剪贴板历史，一键粘贴

---

## 二、宠物互动

### 1. 喂养与养成系统
- 饥饿度、心情值、体力值等多维状态
- 右键喂食（多种食物），影响状态不同
- 长期不喂食宠物变虚弱

### 2. 宠物进化 / 成长
- 陪伴时间越长，幼崽→成年
- 解锁不同外观形态
- 解锁稀有种类（达成特定条件）

### 3. 装扮 / 饰品系统
- 帽子、眼镜、围巾、背饰等
- 通过日常互动解锁
- 拖拽穿戴

### 4. 语音 / 声音互动
- 脚步声、呼噜声、开心叫声
- 对系统声音反应（通知声时转头）
- 可选麦克风输入反应

### 5. 对话气泡系统
- 随机冷笑话/格言
- 时间相关问候（"这么晚了还不睡？"）
- 天气吐槽
- 当前应用调侃

### 6. 小游戏
- 接球：附近弹出小球，点击击中得分
- 猜拳：剪刀石头布对战
- 捉迷藏：宠物躲到屏幕边缘

### 7. 多宠物支持
- 多只宠物同时显示
- 宠物间互动（打架、玩耍、交流）
- 可繁衍后代

### 8. 成就系统
- "第一次抚摸"、"连续陪伴7天"、"深夜守护者"
- 成就解锁饰品或新宠物
- 徽章展示

### 9. 宠物每日日程
- 自主生物钟：晨起伸懒腰、中午觅食、傍晚散步、夜间犯困

### 10. 桌面环境互动
- 感知窗口位置，跳到标题栏上、躲到窗口后面、推开小窗口

## 建议优先级

| 优先级 | 功能 | 理由 |
|--------|------|------|
| 高 | 对话气泡 + 天气联动 | 改动量小、见效快、可玩性强 |
| 高 | 喂养养成系统 | 核心循环，长期留存 |
| 中 | 番茄钟 / 休息提醒 | 实用价值高，利于传播 |
| 中 | 装扮系统 | 可玩性好，解锁驱动 |
| 低 | 小游戏 / 多宠物 | 开发量大，后续扩展 |
